### 1-2. 아나콘다 세팅
- conda env list                        (가상환경 목록 확인)
- conda create -n developer python=3.11 (가상환경 생성)
- conda activate developer              (가상황경 접속)
- pip list                              (설치 된 라이브러리 목록 확인)
- pip install [라이브러리]               (라이브러리 설치)
- cls                                   (화면 clear)
